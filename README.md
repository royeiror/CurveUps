# CurveUp Toolchain

A Windows GUI toolchain for converting 3D objects to 2D fabric patterns, inspired by the CurveUp research paper.

## Features

- Load 3D models (STL, OBJ, PLY, OFF)
- Mesh parameterization and optimization
- Fabric-aware pattern generation
- Export to DXF/SVG for printing
